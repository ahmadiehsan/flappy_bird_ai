# Flappy Bird (NEAT Algorithm)

AI plays Flappy Bird

## Usage

```shell
git clone <this/repo/url>
cd <cloned_dir>

curl -LsSf https://astral.sh/uv/0.11.7/install.sh | sh
uv tool install rust-just

just manage train
```
